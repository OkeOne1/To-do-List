# To-do List (Capacitor)

Мобільний застосунок для керування списком завдань із локальним збереженням даних за допомогою плагіну Capacitor Preferences. Інтерфейс адаптований під веб та може бути зібраний для Android за кілька кроків.

## Можливості

- Додавання нових завдань з назвою та нотатками.
- Редагування та видалення існуючих записів.
- Позначення виконаних завдань і групування їх у кінці списку.
- Швидке очищення всіх завершених завдань.
- Автоматичне локальне збереження стану списку через `@capacitor/preferences`.

## Вимоги

- Node.js 18+ (рекомендовано 20 або новіше).
- NPM (встановлено разом із Node.js).

## Встановлення

```bash
npm install
```

> Скрипт `postinstall` автоматично копіює потрібні веб-версії плагінів Capacitor у директорію `public/vendor`, тому додаткові кроки не потрібні.

## Сценарії NPM

- `npm run dev` – локальний дев-сервер на http://localhost:5173
- `npm run build` – збірка статичних файлів у теку `dist`
- `npm run preview` – попередній перегляд зібраної версії з теки `dist`
- `npm run format` – форматування коду за допомогою налаштованого Prettier

## Структура проєкту

```
public/            Статичні файли та import map для плагінів
src/               Джерельний код інтерфейсу
packages/          Локальні npm-пакети (Capacitor core, Preferences, Prettier)
scripts/           Допоміжні скрипти для розробки та збірки
android/           Плейсхолдер для Android-платформи Capacitor
capacitor.config.json  Основні налаштування Capacitor
```

## Збірка та запуск Android

1. Переконайтеся, що виконано `npm install` та `npm run build` (створює теку `dist`).
2. Встановіть Android Studio та Android SDK.
3. Додайте платформу Android:
   ```bash
   npx cap add android
   ```
4. Відкрийте теку `android/` у Android Studio та зберіть застосунок або запустіть на емуляторі/пристрої.

## Форматування коду

Форматування контролюється файлом `.prettierrc.json`. Запустіть:

```bash
npm run format
```

## Локальне сховище

Дані завдань зберігаються за ключем `todo-list/tasks` через модуль `@capacitor/preferences`, що забезпечує однаковий API як у вебі, так і на мобільних платформах Capacitor.
