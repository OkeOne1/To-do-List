# Android platform scaffold

Ця текова структура зарезервована для Android-проєкту Capacitor. Після встановлення Android Studio запустіть:

```bash
npx cap add android
```

Папка `android/` міститиме повноцінний Android-проєкт для збірки APK. Поточний файл пояснює призначення каталогу у репозиторії.
